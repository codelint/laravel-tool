# laravel-tool
tool to help developing on laravel (main on db operator)

DBOperator have method below, detail see the tests/case/Codelint/Laravel/Database/DBOperatorTest.php 

* findBy
* findById
* findByOne
* findByField
* exist
* insert
* upsert
* upsertById
* add
* updateById
* sum
* rows
* page 
* orderBy
* groupBy 
* leftJoinOn 
* innerJoinOn

=== metadata table === 
* meta($iid, $key, $value = false)
* metadata($iid, $kvs)
